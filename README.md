## Featured Project: HARDAM Queue

**[HARDAM Queue](https://github.com/YakaCnoder/HARDAM-Queue)** — Smart Queue & Citizen Service Management Platform

A queue management platform engineered specifically for schools, universities, and government offices — built and deployed solo, now running live in production across 5+ institutions in Mindanao.

- **Stack:** Java Swing, PHP, MySQL, Python, WebSocket
- **Security:** TOTP 2FA, PBKDF2 hashing, AES-GCM encryption, HMAC-verified payments
- **Results:** 52% reduction in wait time · 79% increase in daily capacity · 44% higher satisfaction

HARDAM Queue is proprietary commercial software. Full product documentation, security policy, and licensing terms live in its own repository — see the [HARDAM Queue Project Repository](https://github.com/YakaCnoder/HARDAM-Queue).
